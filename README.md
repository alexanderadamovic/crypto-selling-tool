# Crypto Sell Tool

A Python program that calculates the percentage change and profit/loss of a cryptocurrency investment, then recommends whether to sell, hold, or stop loss based on user-defined targets.

## Features
- Takes user input for coin, investment amount, buy price, current price, sell target, and stop loss
- Calculates percentage change
- Calculates profit or loss
- Gives a sell, hold, or stop-loss decision

## How to Run
python crypto_sell_tool.py

## Example
Input:
- Coin: BTC
- Amount invested: 1000
- Buy price: 100
- Current price: 110
- Sell target: 5
- Stop loss: 3

Output:
- Decision: Sell
- Profit/Loss: $100.00
- Percentage Change: 10.00%
